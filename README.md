# Fashion Design IP and Collaboration System

A comprehensive blockchain-based system for protecting fashion design intellectual property and managing collaborative relationships between designers and manufacturers.

## Overview

This system provides a decentralized platform for fashion designers to:
- Register and protect their intellectual property
- Manage licensing agreements for design elements and patterns
- Collaborate transparently with manufacturers
- Establish proof of creation dates for patent applications
- Share revenue from collaborative projects

## Smart Contracts

### 1. Design Registry (`design-registry.clar`)
- Register new fashion designs with metadata
- Establish proof of creation timestamps
- Manage design ownership and transfers
- Store design categories and descriptions

### 2. IP Protection (`ip-protection.clar`)
- Protect design intellectual property rights
- Handle IP disputes and claims
- Manage design patent applications
- Track IP usage and violations

### 3. Licensing Manager (`licensing-manager.clar`)
- Create and manage licensing agreements
- Set licensing terms and conditions
- Handle license transfers and renewals
- Track licensed design usage

### 4. Collaboration Hub (`collaboration-hub.clar`)
- Facilitate designer-manufacturer partnerships
- Manage collaborative project proposals
- Track project milestones and deliverables
- Handle collaboration agreements

### 5. Revenue Sharing (`revenue-sharing.clar`)
- Distribute revenue from design sales
- Manage royalty payments to designers
- Handle collaborative project profit sharing
- Track payment history and balances

## Key Features

- **Immutable Design Records**: All designs are permanently recorded on the blockchain
- **Timestamp Verification**: Cryptographic proof of creation dates for patent applications
- **Transparent Licensing**: Clear terms and automated license management
- **Collaborative Workflows**: Structured processes for designer-manufacturer partnerships
- **Automated Revenue Distribution**: Smart contract-based payment splitting

## Getting Started

1. Install dependencies: `npm install`
2. Run tests: `npm test`
3. Deploy contracts using Clarinet
4. Interact with contracts through the provided interfaces

## Testing

The system includes comprehensive tests using Vitest to ensure contract functionality and security.
